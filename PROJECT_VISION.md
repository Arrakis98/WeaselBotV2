# Project Vision

Weasel Bot V2 is a free, self-hosted, interactive Discord music companion.

The core goal is dependable audio playback with a friendly Discord-native interface. The bot should feel useful during normal music sessions and expressive enough to become part of a community's personality.

## What V2 Should Become

- A reliable Lavalink-backed music bot.
- A slash-command-first Discord application.
- A rich Discord UI using embeds, buttons, select menus, and eventually modals.
- A playlist and listening-history companion.
- A user-profile aware music system.
- A local-library friendly bot that can play music from a read-only mounted directory.
- A safe migration target for old JSON playlist and history exports.
- A playful bot with optional personality features.
- A future host for Chaos / Mad DJ mode.

## Non-Goals

- No required paid API.
- No required cloud service.
- No required AI service.
- No secrets in Git.
- No private Arcadia infrastructure in the public repository.

AI and Ollama integrations may be explored later, but they must remain optional and outside the core build.
